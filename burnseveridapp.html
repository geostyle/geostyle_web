Burn SeveridApp


Version 1.0 (2023-02-08)
Author: Andrés Salazar (ajsalazar@uc.cl)


Web application to obtain burn severity data. These data are extracted from Landsat 8, so the images are available since 2013.

The Normalized Burn Ratio (NBR) is an index designed to highlight burnt areas in large fire zones.The difference between the pre-fire and post-fire NBR obtained from the images is used to calculate the delta NBR (dNBR or ∆NBR), which then can be used to estimate the burn severity. The images are obtained every 16 days, but as this app was built in GEE, it may take longer to be available. If you get an error using the app, it is likely that the images are not available for the dates you chose.
